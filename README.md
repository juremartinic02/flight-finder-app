# Flight Finder App

This project contains the backend and frontend code for a Flight Finder application.

## Structure

- \backend/\ — Spring Boot backend REST API
- \frontend/\ — React frontend application created with Vite

## How to run

### Backend

1. Navigate to /backend
2. Run the Spring Boot app (e.g., via your IDE or \./mvnw spring-boot:run\)

### Frontend

1. Navigate to /frontend
2. Run \npm install\ (if you haven’t already)
3. Run \npm run dev\ to start the React development server

## Notes

- CORS is configured in the backend to allow frontend requests.
- The frontend uses dummy data until connected to the backend API.
